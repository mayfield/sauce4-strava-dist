How it works...

Sauce Performance is built on an advanced synchronization system that imports an athlete's entire activity history from Strava servers to your browser's internal storage. It uses proven browser technology to store, index and analyze the activities without the use of 3rd party servers. Once an athlete's activity history is synchronized, a world of new activity analysis possibilities are available!

FAQ

Does it cost anything?
Basic functionality works for free, but you can unlock the full power for $2/month via Patreon.
How does Sauce get activity data?
Sauce uses the same servers and webpages that the normal Strava website uses. No special API access or 3rd party servers are used.
Why does the sync process take so long the first time?
If activity data is imported too quickly, Strava will temporarily rate-limit your account. Sauce is programmed to stay safely within these limits as long as you only sync one device at a time.
Why does Sauce need to "Synchronize" my data for the Performance features?
To provided advanced training and historical data access, it is necessary to have direct access to all data in one place. Sauce Performance makes use your browser's own storage facility to do this.
Where is all this data being stored?
In your web browser's storage system called "IndexedDB." This is a built-in database that, for most platforms, is capable of storing and querying gigabytes of data efficiently. Data stays with you on your device.
How do I protect my data?
Backups, backups, backups! Use the "Export Data" feature in the sync control panel to create backup files that you can place somewhere safe.
